# Project: SO-101 Arm

Manipulation driven by learned policies: teleoperate the SO-101 arm,
collect demonstration datasets, train imitation-learning policies, and
run them on real hardware. 

## Milestones

1. Teleoperation and dataset recording working end to end
2. A trained policy completes a pick task autonomously
3. Policy generalizes to object positions not in the training data
