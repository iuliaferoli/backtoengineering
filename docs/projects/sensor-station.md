# Project: Sensor Station

Read the physical world: wire up sensors (temperature, distance, IMU),
stream their readings into Python, and visualize live data. The
graduation step between making an output (blinking an LED) and making a
system that senses and reacts.

## Milestones

1. One sensor wired and read reliably from code
2. Readings logged and plotted over time
3. A threshold triggers a physical output
