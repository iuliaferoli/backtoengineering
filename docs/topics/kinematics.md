# Robot Kinematics

Kinematics studies the motion of robot manipulators without considering forces, covering forward and inverse kinematics for position control.

## You can move on when you can...

- Calculate forward kinematics using DH parameters
- Solve inverse kinematics for robot arms
- Understand workspace and singularities
- Apply transformation matrices
- Implement basic trajectory planning
