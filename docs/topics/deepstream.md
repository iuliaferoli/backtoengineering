# DeepStream SDK

NVIDIA DeepStream SDK is a framework for building AI-powered video analytics applications with optimized multi-stream processing.

## You can move on when you can...

- Build video analytics pipelines with GStreamer
- Deploy multi-camera inference applications
- Integrate custom AI models into DeepStream
- Configure metadata extraction and tracking
- Optimize for real-time multi-stream processing
