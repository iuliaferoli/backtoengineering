# AI Agents & MCP

AI agents are applications that can reason about tasks, make decisions, and use external tools to accomplish goals. Unlike a standalone large language model (LLM) that only generates responses from its available context, an AI agent can interact with external systems such as APIs, databases, search tools, or other services to complete multi-step tasks.

The Model Context Protocol (MCP) is an open protocol that provides a standardized way for AI applications to connect to tools and data sources.

## You can move on when you can...

- Explain the difference between an LLM and an AI agent
- Build a small agent that calls at least one external tool
- Describe the purpose of the Model Context Protocol (MCP)
- Explain when MCP is useful for connecting tools and data
- Understand when an AI agent is more appropriate than using an LLM alone
