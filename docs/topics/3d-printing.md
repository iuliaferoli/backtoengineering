# 3D Printing

Turning CAD models into physical parts: slicing, tolerances, and materials.
You can find loads of open source STL files to download and print at home.

## You can move on when you can...

- Print a part that fits another part (tolerance test)
- Diagnose a failed print (adhesion, stringing, layer shift)

## Open source model repositories
- [Thingiverse](https://www.thingiverse.com/)
- [Printables](https://www.printables.com/)
- [Makerworld](https://makerworld.com/)
- [Simple 3D arm model](https://cults3d.com/en/3d-model/gadget/grippy-bot)
