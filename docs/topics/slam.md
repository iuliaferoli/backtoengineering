# SLAM (Simultaneous Localization and Mapping)

SLAM enables robots to build a map of an unknown environment while simultaneously tracking their location within it.

## You can move on when you can...

- Understand sensor fusion (LiDAR, cameras, IMU)
- Implement particle filters or EKF SLAM
- Work with occupancy grids and point clouds
- Recognize loop closure and place recognition
- Deploy SLAM in ROS 2 (Nav2, RTAB-Map, ORB-SLAM)
